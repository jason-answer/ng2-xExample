Root Router

In Project, app.routes.ts is application's root router.

Lazy loading modules with the Router

Use lazy loading moudule configure to router.

{{'{ path: 'examples', loadChildren: './example/example.module#ExampleModule' }}''}

Configuration @NgModule: In the ExampleModule, imports ExampleRoutingModule to ExampleModule of @NgModule. The ExampleRoutingModule is routes for child component. All child component should be add [declarations] to parent module, NOT to be add App.module

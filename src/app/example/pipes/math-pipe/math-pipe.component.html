Math Pipe

min

Returns the minimum of a given array

Usage: array | min

{{ [1, 2, 3, 1, 2, 3] | min }}

Output: {{ [1, 2, 3, 1, 2, 3] | min }}
max

Returns the maximum of a given array

Usage: array | max

{{ [1, 2, 3, 1, 2, 3] | max }}

Output: {{ [1, 2, 3, 1, 2, 3] | max }}
sum

Returns the sum of a given array

Usage: array | sum

{{ [1, 2, 3, 4] | sum }}

Output: {{ [1, 2, 3, 4] | sum }}
percentage

Returns percentage between numbers

Usage: number | percentage: [total | default = 100]: [floor | default = false]

{{ 5 | percentage }}

Output: {{ 5 | percentage }}

{{ 10 | percentage: 160 }}

Output: {{ 10 | percentage: 160 }}

{{ 5 | percentage: 160: true }}

Output: {{ 10 | percentage: 160: true }}
